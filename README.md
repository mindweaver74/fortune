# 🔮 운세가 궁금하지?

> 생년월일로 보는 오늘의 운세 카드 — 매일 바뀌는 나만의 총운·금전운·애정운·건강운 + 행운의 로또 번호

**▶ 바로 사용하기: https://mindweaver74.github.io/fortune**

<br>

## ✨ 소개

생년월일 하나만 넣으면 오늘 하루의 운세를 한 장의 카드로 보여주는 무료 웹 도구입니다.
신비로운 카드를 탭해서 뒤집으면 오늘의 운세가 공개됩니다.

- **오늘의 총운** — 100점 만점 점수와 4단계 등급 (대길 🌟 / 길 😊 / 보통 😌 / 주의 🌧️)
- **분야별 운세** — 💰 금전운 · ❤️ 애정운 · 🍀 건강운, 별점과 한 줄 조언
- **행운의 로또 번호** — 생년 천간의 오행 상생 원리로 가중치를 준 6개 번호
- **이미지 저장** — 결과 카드를 인스타그램 세로형(1080×1350) PNG로 저장해 공유

## 🎯 특징

- **매일 바뀌는 결정론적 운세**: 생년월일 + 오늘 날짜(+이름)를 시드로 사용해, 같은 사람은 하루 종일 같은 결과가 나옵니다. 자정이 지나면 자동으로 새 운세!
- **양력/음력 지원**: 사주 기준에 맞춰 양력 선택 시 입춘(2월 4일경) 이전 출생은 전년도 천간을 자동 적용합니다.
- **완전 오프라인 단일 파일**: 외부 라이브러리·서버 통신 없이 HTML 파일 하나로 동작합니다. 이미지 생성도 순수 Canvas API로 처리합니다.
- **개인정보 안전**: 입력한 생년월일과 이름은 브라우저에만 저장되며 어디로도 전송되지 않습니다.
